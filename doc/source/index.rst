Welcome to ``diaGrabber``'s documentation!
=========================================
``diaGrabber`` is a module to read, process and plot n-dimensional-data fully written in `Python <http://www.python.org/>`_.






Download the source
==================
You can ``diaGrabber`` on it's `sourceforge-site <http://www.ooooooooooooooooo.org/>`_.

There is also a git-link to browse the code online.

Installation
============

To install ``diaGrabber`` on your system go to the main-'diaGrabber'-Folder and type in a termial:

... **in Linux**::

   sudo python setup.py install

... **in Windows**::

   ?????????

Use diaGrabber via::

   import diaGrabber

in a Python script or shell.
